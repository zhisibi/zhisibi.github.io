这个方案如何解决你的问题？

  1. 路径自动修正：当执行带有 _config.zzac.yml 的命令时，Hexo 会把所有的内部资源链接（如 /css/main.css）自动重写为 /blog/css/main.css。这样你在点击博文时，URL 就会变成 .../blog/2025/12/31/...，从而被 Caddy 准确捕获。

  2. 避免误伤：在发布 GitHub 时，脚本只用了原配，所以 GitHub 上的页面链接依然是干净的根路径。
